App Data

This information serves to inform you on how to edit this app and add your own pages.


app.json

The app.json file is used to configure the app. You can change the App Name, Icon, version, theme, font, and add custom Pages.


The pages is an array of objects with the following properties:
- id: The id of the page
- name: The name of the page that appears on the menu
- icon: [optional] The icon of the page that appears on the menu
- html: The html Structure of the page
- style: [optional] The style of the page
- script: [optional] The script of the page



		{
			"name": "App Template",
			"icon": "/icons/app-icon.png",
			"version": "1.0.0",
			"defaultPage": "home",
			"defaultTheme": "dark",
			"defaultFont": "Nunito",

			"displayMenu": true,
			"displayHome": true,
			"displaySettings": true,

			"allowInstall": true,

			"pages": [
				{
					"id": "appinfo",
					"name": "App Info",
					"html": "/pages/appinfo.html",
					"style": "/pages/appinfo.css",
					"script": "/pages/appinfo.js",
					"icon": "/pages/question-white.png"
				},
				{
					"id": "appdata",
					"name": "App Data",
					"html": "/pages/appdata.html",
					"style": "/pages/appdata.css",
					"script": "/pages/appdata.js"
				}
			]
		}

If you would like your page assets to be available offline you can add them to the ASSETS_CUSTOM array in the service-worker.js file.
